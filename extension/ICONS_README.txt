
⚡ Amplify Extension Icons
========================
For production, you would need actual PNG icons:
- icon16.png (16x16)
- icon48.png (48x48) 
- icon128.png (128x128)

You can create these using:
1. Canva, Figma, or any design tool
2. Use the ⚡ lightning bolt symbol
3. Purple/pink gradient background
4. Export as PNG files
